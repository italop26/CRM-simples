# CRM-simples
# Sistema CRM em Django

Este projeto é um sistema CRM (Gerenciamento de Relacionamento com o Cliente) desenvolvido com Django para praticar conceitos de desenvolvimento web back-end e front-end.
O sistema possui controle de usuários, produtos, vendas e estoque, simulando o funcionamento de um pequeno sistema comercial.



## 🚀 
* Cadastro de usuários
* Cadastro de produtos
* Visualização de usuários
* Controle do estoque
* Relatórios

### Vendedor

* Visualização dos produtos
* Registro de vendas
* Baixa automática do estoque após a venda

### Estoquista

* Visualização do estoque
* Atualização da quantidade de produtos

### Caixa

* Registro das entradas e saídas do dia

## ▶️ Como executar o projeto

1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

2. Entre na pasta

```bash
cd SEU-REPOSITORIO
```

3. Instale as dependências

```bash
pip install -r requirements.txt
```

4. Execute as migrações

```bash
python manage.py migrate
```

5. Inicie o servidor

```bash
python manage.py runserver
```

6. Abra o navegador

```
http://127.0.0.1:8000/
```
